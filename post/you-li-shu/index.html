【初中数学】六、有理数的加法

2021-01-18

一、有理数的加法

  • 同号的两个有理数相加，取相同的符号，并把 绝对值 相加。

    例如：+3 + +4 = 7；-3 + -4 = -7.

  • 符号不同的两个有理数相加，若绝对值不相等的情况下，取绝对值较大的数的符号，并用 绝对值 大的数取减去绝对值小的获取结果。

    例如：+7 + -3 = +4；-7 + +4 = -3.

  • 符号不同的两个有理数相加，若两数绝对值相等，则结果为0。

    例如：+2 + -2 = 0.

  • 一个数同0相加，结果仍然是这个数。

    例如：-2 + 0 = -2；+2 + 0 = +2.

二、有理数加法的交换律、结合律

有理数运算中，加法的交换律与结合律仍适用。

  • 加法交换律
    两个数相加，交换加数的位置，和不变。
    例如：a + b = b + a。
  • 加法结合律
    三个数相加，先把前两个数相加，或者先把后两个数相加，和不变。
    例如：(a + b) + c = a + (b + c)。

一般的，任意若干个数相加，无论各数相加的先后次序如何，其和不变。
