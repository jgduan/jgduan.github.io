【初中数学】七、有理数的减法

2021-01-19

学习目标

  1. 理解掌握有理数的减法法则；
  2. 会进行有理数的减法运算；
  3. 能够把有理数的减法运算转化为加法运算，进而写成省略括号和加号的和的形式。

一、样例计算

  1. 4 - 2 = 2; 4 + (-2) = 2;
  2. 4 -1 = 3; 4 + (-1) = 3;
  3. 4 - 5 = -1; 4 + (-5) = -1.
  4. 4 - 0 = 4; 4 + 0 = 4;
  5. 4 - (-1) = 5; 4 + 1 = 5;
  6. 4 - (-2) = 6; 4 + 2 = 6.

通过以上运算，得出结论：减去一个数，等于加上这个数的相反数，也可以表示为：a - b = a + (-b)。

注意：减法在运算时有2个要素要发生变化。
1.减号变成加号；
2.减数变成相反数；
如：a - b = a + (-b)。

二、例题

  1. 计算 (-20) + (+3) - (-5) - (+7)
    解： (-20) + (+3) - (-5) - (+7)
    = (-20) + (+3) + (+5) + (-7)
    = [(+3) + (+5)] + [(-20) + (-7)]
    = (+8) + (-27)
    = -19
  2. 计算(-72) - (-37) - (-22) - 17
    解：(-72) - (-37) - (-22) - 17
    = (-72) + (+37) + (+22) + (-17)
    = [(-72) + (-17)] + [(+37) + (+22)]
    = -89 + (+59)
    = -30

三、有理数的减法——省略括号和加号

例子：一架飞机在进行特技表演，起飞后的高度变化如下：

  1. 上升4.5千米，记作+4.5千米；
  2. 下降3.2千米，记作-3.2千米；
  3. 上升1.1千米，记作+1.1千米；
  4. 下降1.4千米，记作-1.4千米。
    问，此时飞机比起飞点高了多少千米？

解法1：
+4.5 + (-3.2) + (+1.1) + (-1.4)
= [(+4.5) + (+1.1)] + [(-3.2) + (-1.4)]
= +5.6 + (-4.6)
= 1

解法2：
4.5 - 3.2 + 1.1 -1.4
= 1.3 + 1.1 - 1.4
= 2.4 - 1.4
= 1

解法2对比解法1，省略了加号和括号，两者的计算结果是一致的，但解法2看起来更简单明了一点。
在这里，我们把4.5 - 3.2 + 1.1 - 1.4看作为4.5，-3.2，1.1，-1.4的和，也叫代数和。

代数和是指两个或更多的数或量按照代数加法规律取符号(如 +或-)的总和。
